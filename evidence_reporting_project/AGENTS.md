# AGENTS

This project is organized around the paper workflow:

- same-day evidence is always the primary basis
- retrieved knowledge cards are background support only
- reports must follow a fixed schema
- key numerical statements must be audited after generation
- Mode C is reserved for future extension only

When editing the project:

- keep intermediate outputs on disk
- do not collapse retrieval, prompting, and auditing into a single file
- do not silently fill missing evidence
- do not implement fake visual inference for Mode C
